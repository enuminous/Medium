1. Quantum Non-Local Computation (QNC) Implementation

Python Code for Non-Local Computation with Entangled Qubits (Qiskit Example)

from qiskit import QuantumCircuit, Aer, transpile, execute
from qiskit.providers.aer import AerSimulator
from qiskit.quantum_info import state_fidelity
import numpy as np

# Create a quantum circuit for entanglement
qc = QuantumCircuit(2)
qc.h(0) # Hadamard gate to create superposition
qc.cx(0, 1) # CNOT gate to entangle qubits

# Measurement for verification
qc.measure_all()

# Simulate on a quantum backend
simulator = AerSimulator()
compiled_circuit = transpile(qc, simulator)
result = execute(compiled_circuit, simulator).result()
counts = result.get_counts()

print("Quantum Non-Local Computation Results:", counts)

Python Code for Quantum Error Correction

from qiskit import QuantumCircuit, Aer, transpile, execute
from qiskit.providers.aer.noise import NoiseModel
from qiskit.providers.aer.utils import approximate_quantum_error

# Define a noise model
noise_model = NoiseModel()
bit_flip_error = approximate_quantum_error([[1, 0], [0, 0]], 0.05)
noise_model.add_all_qubit_quantum_error(bit_flip_error, 'id')

# Create a quantum error correction circuit
qc = QuantumCircuit(3, 3)
qc.h(0)
qc.cx(0, 1)
qc.cx(0, 2)
qc.measure([0, 1, 2], [0, 1, 2])

# Simulate the circuit with noise
simulator = AerSimulator()
compiled_qc = transpile(qc, simulator)
result = execute(compiled_qc, simulator, noise_model=noise_model).result()
print("Error Correction Results:", result.get_counts())

Quantum Key Distribution (QKD) with Entangled Qubits

This implementation follows the BB84 Protocol, where two parties (Alice and Bob) share an entangled qubit pair to establish a quantum-secure key.

Python Code for Entanglement-Based Key Exchange

from qiskit import QuantumCircuit, Aer, transpile, execute
from qiskit.visualization import plot_histogram

# Quantum circuit for entangled key distribution
qc = QuantumCircuit(2, 2)

# Step 1: Create an entangled Bell pair
qc.h(0) # Hadamard gate to create superposition
qc.cx(0, 1) # CNOT gate to entangle qubits

# Step 2: Measurement in the computational basis
qc.measure([0, 1], [0, 1])

# Simulate the quantum key exchange
simulator = Aer.get_backend('qasm_simulator')
compiled_qc = transpile(qc, simulator)
result = execute(compiled_qc, simulator, shots=1024).result()

# Retrieve key distribution results
key_distribution = result.get_counts()

print("Entanglement-Based Quantum Key Distribution Results:")
print(key_distribution)

# Plot histogram of key exchange outcomes
plot_histogram(key_distribution)

✔ Purpose:

Alice and Bob share a perfectly correlated key derived from entangled qubits.

No classical key exchange occurs, preventing interception by eavesdroppers.

Keys are generated using Bell state measurements, ensuring post-quantum security.
