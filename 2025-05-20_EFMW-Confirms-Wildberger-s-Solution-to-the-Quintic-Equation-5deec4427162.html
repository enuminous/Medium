A New Kind of Answer

Let’s revisit the old boundary.

A general quintic equation looks like this:

P(x) = x⁵ + a₄x⁴ + a₃x³ + a₂x² + a₁x + a₀ = 0

The radical-based methods that solved quadratics, cubics, and quartics fail here because of the symmetry groups involved. Galois theory proves that there's no universal formula using roots for quintics and beyond.

Wildberger's audacious claim is this: stop using radicals. Stop trying to “solve” in the old sense. Instead, build the answer recursively, using pure rational sequences derived from combinatorics.

Instead of a square root of a root, we get something different:

r = ∑ cₙ(a₀, a₁, ..., a₄) · xⁿ, for n = 0 to ∞

These coefficients, cₙ, are determined by a multidimensional extension of the Catalan numbers—famous for counting bracketed expressions, polygon triangulations, and recursive tree structures.

Wildberger calls the new array of values “Geodes.”

Why This Isn’t Cheating: Galois vs. Geodes

To be clear: this doesn’t violate Galois theory.

Galois said you can’t express quintic solutions with radicals.
Wildberger never tried to.

Instead, he treats the solution as a computable infinite series — much like how we represent sine, cosine, or exponential functions. If we accept Taylor series as valid representations of transcendental functions, why not polynomials?

This is not breaking the rule. It is rewriting the question.
